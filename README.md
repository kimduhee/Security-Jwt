JPA
------------
> 기본키 자동 생성 전략 4가지
+ IDENTITY
```
기본 키 생성을 데이터베이스에 위임한다.
```
+ SEQUENCE
```
데이터베이스 Sequence Object를 사용하는 것으로 DB가 자동으로 숫자를 generate 한다.
```
+ TABLE
```
키 생성 전용 테이블을 만들어서 데이터베이스 시퀀스 처럼 생성한다.
```
+ AUTO
```
기본 설정값으로 위 세 가지 전략을 자동으로 지정한다.
```
